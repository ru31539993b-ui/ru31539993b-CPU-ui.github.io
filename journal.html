📖 電腦課心得

作者：幼保三丙陳珮榛81410360 ｜ 日期：2026 年 6 月 15 日

從害怕電腦到喜歡電腦課

老實說，開學前我看到課表上有「電腦課」三個字，心裡是有點害怕的。我對電腦的印象就是打報告和上網，那些程式碼之類的東西感覺離我很遙遠。但是這學期上完這門課，我的想法完全改變了。

老師從最基本的 HTML 開始教，說網頁就是用標籤一層一層把內容包起來。一開始覺得那些小於大於的符號很陌生，但練習幾次之後，發現只要記住幾個常用的標籤，就能自己做出一個有標題、有段落、還有圖片的網頁。當我第一次在瀏覽器裡看到自己打出來的成果時，真的很有成就感。

CSS 讓網頁變漂亮

學會 HTML 之後，老師教我們 CSS——幫網頁打扮化妝的工具。原本黑白的文字加上顏色、字型、邊框和陰影之後，整個網頁看起來完全不一樣了。我花了最多時間在調整顏色搭配，因為我希望我的網頁看起來溫暖又可愛，就像幼稚園教室的布置一樣。老師說這叫做「使用者體驗」，要讓來看網頁的人覺得舒服。我覺得這對我以後要從事的幼教工作來說，也是很重要的觀念。

JavaScript 讓網頁活起來

最讓我驚豔的是 JavaScript。老師說 HTML 是靜態的，JavaScript 可以讓網頁跟使用者互動。我們學了變數、判斷式、迴圈和函式，雖然有時候邏輯轉不過來，但老師總是用生活化的例子解釋，比如用「如果下雨就帶傘」來說明 if 判斷式。當我第一次成功寫出一個會回應的按鈕時，我興奮地找隔壁同學來看，那種開心到現在還記得。

分組合作與互相幫助

這堂課讓我印象深刻的還有分組實作。我和同學一起討論版面怎麼排、顏色怎麼配，遇到 bug 的時候互相幫忙找問題。有時候只是少了一個分號或括號，程式就跑不動了，這時候兩個人一起看，往往比一個人埋頭苦找快得多。我學到寫程式不只是一個人坐在電腦前面，團隊合作也同樣重要。

對未來的啟發

我是幼保科的學生，以前從沒想過電腦程式跟我有什麼關係。但這學期的電腦課讓我發現，未來我在幼兒園工作時，也可以自己做教學用的互動教材、做班級網頁跟家長溝通。電腦不是只有工程師才需要學，它是一個工具，可以幫助我們在任何領域做得更好。

謝謝老師這學期的教導，讓我從害怕電腦到現在敢自己動手嘗試。雖然我的作品還很簡單，但這只是一個開始，以後我還會繼續學習更多東西！
