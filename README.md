# 🎮 兒童遊戲樂園

幼保三丙陳珮榛81410360 的電腦課期末作品 — 一個純靜態的兒童遊戲入口網站。

## 網站頁面

| 頁面 | 說明 |
|------|------|
| `index.html` | 遊戲選單首頁 |
| `memory.html` | 4×4 記憶翻牌配對（動物 emoji） |
| `math.html` | 加減法算術練習（三種難度） |
| `puzzle.html` | 3×3 滑塊拼圖（8 格動物 emoji） |
| `guide.html` | 遊戲玩法說明 |
| `journal.html` | 電腦課學習心得 |

## 使用方式

直接雙擊 `index.html` 用瀏覽器開啟即可，無需安裝任何軟體或套件。

## 技術架構

- 純 HTML + CSS + JavaScript，無框架、無建置工具、無相依套件
- 共用樣式表：`style.css`
